# ayto-monitor
# Monitor de citas Ayuntamiento Fuenlabrada

Monitor automático que consulta disponibilidad de citas y avisa mediante ntfy.

## Configuración

Crear un secreto:

Settings → Secrets → Actions

Nombre:

NTFY_TOPIC

Valor:

tu-tema-ntfy

## Ejecución manual

Actions → Monitor citas → Run workflow

## Frecuencia

Cada 5 minutos.

## Funcionamiento

Solo envía aviso cuando aparece disponibilidad nueva.# Updated
